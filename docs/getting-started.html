ğŸ“¥ Installation

Step 1: Download Lodestar Suite

  • Go to the Releases page
  • Download the latest lodestar-suite-1.21.8-*.jar file
  • Save it somewhere you can find it (like your Desktop)

Step 2: Install Fabric Loader

  • Download Fabric Loader for Minecraft 1.21.8
  • Install it using the Fabric installer
  • Make sure you have Java 21 or later installed

Step 3: Install Lodestar Suite

  • Open your Minecraft launcher
  • Select the Fabric profile (it should appear after installing Fabric)
  • Click "Play" to start Minecraft once (this creates the mods folder)
  • Close Minecraft
  • Open your Minecraft folder:

- Windows: Press Win + R, type %appdata%\.minecraft, press Enter - Mac: ~/Library/Application Support/minecraft - Linux: ~/.minecraft

  • Open the mods folder
  • Copy the lodestar-suite-1.21.8-*.jar file into the mods folder

Step 4: Launch and Enjoy!

  • Start Minecraft with the Fabric profile
  • You should see "Lodestar Suite" in the title screen credits
  • Press Right Shift (default) to open the Lodestar Suite menu

Common Issues

"Lodestar Suite doesn't appear in Minecraft"
  • Make sure you're using the Fabric profile
  • Check that the JAR file is in the mods folder
  • Verify you have Java 21 or later
"Game crashes on startup"
  • Remove other mods temporarily to test
  • Check the Minecraft logs for error messages
  • Make sure you have the latest Fabric Loader
"Menu doesn't open"
  • Try pressing Right Shift
  • Check if the keybind was changed in settings
  • Restart Minecraft
